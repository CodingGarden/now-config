# now-config

A simple CLI to help generate a [now.sh](https://now.sh/) configuration file.

Based on the examples [here](https://github.com/CodingGarden/deploy-with-now-v2).

### Installation

```sh
npm install -g now-config
```

### Usage

```sh
now-config
```
